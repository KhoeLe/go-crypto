📊 API Endpoints

GET /api/v1/health
Health check endpoint
curl http://localhost:8080/api/v1/health
GET /api/v1/price/{symbol}
Get current price for a trading symbol
curl http://localhost:8080/api/v1/price/BTCUSDT
GET /api/v1/ticker/{symbol}
Get 24hr ticker statistics
curl http://localhost:8080/api/v1/ticker/ETHUSDT
GET /api/v1/klines/{symbol}?interval=15m&limit=100
Get historical kline data
curl "http://localhost:8080/api/v1/klines/BTCUSDT?interval=4h&limit=50"
GET /api/v1/indicators/{symbol}?interval=15m
Get technical indicators (RSI 6/12/24, MA 7/25/99, KDJ)
curl "http://localhost:8080/api/v1/indicators/ETHUSDT?interval=15m"
GET /api/v1/analysis/{symbol}?interval=15m
Get complete technical analysis with signals
curl "http://localhost:8080/api/v1/analysis/BTCUSDT?interval=4h"
GET /api/v1/multi-analysis/{symbol}
Get multi-timeframe analysis (15m, 4h, 1d)
curl http://localhost:8080/api/v1/multi-analysis/BTCUSDT
GET /api/v1/signals/{symbol}
Get trading signals
curl http://localhost:8080/api/v1/signals/ETHUSDT
GET /api/v1/symbols
Get supported symbols and timeframes
curl http://localhost:8080/api/v1/symbols

🛠️ Usage Examples

JavaScript/Node.js

// Get BTC analysis fetch('http://localhost:8080/api/v1/analysis/BTCUSDT?interval=4h') .then(response => response.json()) .then(data => { if (data.success) { console.log('BTC Price:', data.data.price.price); console.log('RSI Values:', data.data.rsi); console.log('Signals:', data.data.signals); } });

Python

import requests # Get multi-timeframe analysis response = requests.get('http://localhost:8080/api/v1/multi-analysis/ETHUSDT') data = response.json() if data['success']: for timeframe, analysis in data['data']['analyses'].items(): print(f"{timeframe}: RSI_12 = {analysis['rsi']['RSI_12']}")

curl

# Get indicators with custom timeframe curl -X GET "http://localhost:8080/api/v1/indicators/BNBUSDT?interval=1d" \ -H "Content-Type: application/json" | jq '.'

📋 Supported Assets

Symbols: BTCUSDT, ETHUSDT, BNBUSDT, ADAUSDT, SOLUSDT

Timeframes: 1m, 3m, 5m, 15m, 30m, 1h, 2h, 4h, 6h, 8h, 12h, 1d, 3d, 1w

Indicators:

  • RSI: 6, 12, 24 periods
  • Moving Average: 7, 25, 99 periods (SMA/EMA)
  • KDJ: Stochastic oscillator
  • Volatility: 20-period standard deviation
